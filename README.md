# 경기데이터 분석

현 경기 전반 분석

## 📌 프로젝트 개요
- 목적: 경제 공부
- 기간: 2025.06.28 ~ 2025.07.19

## 🛠 사용 기술
Python, Numpy, Pandas

## 📊 주요 결과
- 성공적인 결과물을 만들지 못함.

## 실패 원인
- 데이터 수집을 제대로 못함.
- 수집했던 데이터 역시 기관에서 분석했던 데이터였기에 인사이트를 도출하지 못했음.

## 🗂 폴더 구조
```
my-project/
├── README.md        ← 가장 중요
├── notebooks/       ← Jupyter notebook (.ipynb)
├── src/             ← 파이썬 스크립트
├── data/            ← 샘플 데이터 (전체 X, 일부만)
└── requirements.txt ← 사용한 라이브러리 목록
```

## 🚀 실행 방법
```bash
pip install -r requirements.txt
python main.py
```
